# ft_printf
Recreation of the printf() function from libc.

| prototype |
| --------- |
| int  *ft_printf(const char *, ...); |

- Supported conversions: `%` `c` `s` `p` `i` `d` `u` `x` `X`
## Usage
| command | info |
| ------- | ---- |
| `make` | compile *libftprintf.a* (std. version) |
|`make fclean`| clean your file |
